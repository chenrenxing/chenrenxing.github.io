流程

webpack.js output.path:指定输出文件的路径。
webpack输出真实的文件，而webpack-dev-server输出的文件只存在于内存中,不输出真实的文件！
output: {
    path: './dist/js/',  //路径
    publicPath: '',  //网站运行时访问的路径
    filename: '[name]-[hash].js' // 生成文件名  name为entry下的属性名

},
webpack-dev-server生成的文件在内存中也是存在dist/js 这个文件内。
