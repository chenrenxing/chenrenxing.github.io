API_List类型

  • List.isList()判断是否是List类型

    const list=List([1,2,3,4])
List.isList(list)

  • List.of()创建新的List

    const l=List.of(1,2,3,4)
 List[1,2,3,4]

  • size获取List的长度

    console.log(l.size)

  • set方法用来设定指定下标的值set(下标，值)
    -1代表从后面往前面

    L3=L.set(0,'122')

  • delete 删除指定下标的值 delete(下标)

  • insert 差入值 insert(下标，值)

  • updata用来更新指定下标的值

    const  a=List([1,2,3,4])
const  b=a.updata(1,x=>x+100)
涉及业务，对下标为1的数据 进行加100

  • clear 清空并返回一个长度为0的新数组

    const c1=List([1,2,3,4])
const c2=C1.clear();
c1还是存在不变，c2为空了

  • push pop unshit shift同数组的同名方法

  • setSize 重新设定数组长度，小与原数组长度会截断，大于原数组长度会哟个underfined进行填充

    const set1=List([1,2,3,4,5,6])
const set2=set1.setSize(2)
截取出来List[1,2]

  • setIn()用来设定嵌套结构的值([第一层下标，第二层下标，..第N层下标]，值) 同理还有deleteIn,insertIn,updateIn

    const arr1=List([List([1,2,3,4]),
                [List([11,22,33,444)],
                [List([111,222,333,444])]])
const arr2=arr1.setIn([2,1],0)会改变222为0

  • concat链接List concat(List1,List2,…ListN)

    const list1=List([1,2,3,4])
const list2=List([11,223,4])
const list3=List([333,4444,4])
const totalList=list1.concat(list2,list3)

  • merge是concat的别名

    const list1=List([1,2,3,4])
const list2=List([11,223,4])
const list3=List([333,4444,4])
const totalList=list1.merge(list2,list3)
  • map 同原生map
  • fliter 同原生的filter

  • flatten 扁平化list

    const f1=List([List([1,2,3,4]),
                [List([11,22,33,444)],
                [List([111,222,333,444])]])
const f2=f1.flatten(Blean/number);
true:代表浅拉平第一层
f2:List=[1,2,3,4,11,22,33,444,111,222,333,444]
flase:代表深N层全部拉平 
number：代表第二次2层结构拉平

  • find 查找，返回第一个符合的结果

    const names=List(['藏三','里斯'，'张丝'])
const names2=names.finds((v,k))=>v.indexOf('张'）!== -1)
console.log(name2)=>藏三
  • findLast 查找，返回最后一个符合的结果 同理find

  • keys 返回所有的下标

    const names=List(['藏三','里斯'，'张丝'])
const keys=names.keys();->迭代器
for(const k of names){
    console.log(k)
    0,1,3,4
}

    //values返回所有的值

    const names=List(['藏三','里斯'，'张丝'])
const values=names.values()
for(const v of names){
    console.log(v)
    '藏三','里斯'，'张丝'
}

    //entries 返回所有entry,(key，value)形式

const names=List(['藏三','里斯'，'张丝'])
const entrys=names.values()
for(const entry of entrys){
    console.log(entry)
    [0,'藏三'],[0,'里斯'],[0,'张丝']
}
