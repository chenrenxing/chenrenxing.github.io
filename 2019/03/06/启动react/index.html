启动react

发表于 2019-03-06

初始化

npm install -g create-react-app
create-react-app my-app
npm start

运行npm run eject生成配置文件，报git的错
主要问题是脚手架添加.gitgnore文件，但是却没有本地仓库，按照以下顺序就可以正常使用
git init
git add .
git commit -m 'manager'
npm run eject

安装所需要的插件

安装react-router、Axios(在4.0版本后建议安装react-router-dom）
安装AntD
暴露webpack配置
安装less-loader
修改less-loader    

这里里采用的是yarn安装更快捷点
yarn add  react-router-dom axios less-loader less
暴露webpack（yarn run eject）后，需要在config/webpack.config.js里面进行less配置，复制一套sass-loader修改下就可以。
yarn add babel-plugin-import 实现antD按需加载
config/webpack.config.js文件中
按照andt官网引入代码loader: require.resolve('babel-loader')在这个模块里
  • 引入到test: /.(js|mjs|jsx|ts|tsx)$/配置里面->plugins下面
  • 这里需要注意的是less版本过高 antd引进的按需加载 （”style”: “css” // style: true 会加载 less 文件）会报错
  • 降低less版本-yarn add less@^2.7.3
import * as xxx from xxx和不带*as(即import xxx from xxx)的区别
react-route-dom函数跳转
