cordova-vue-app

发表于 2018-10-23

Cordova创建应用

[https://cordova.apache.org](https://cordova.apache.org "cordova官网")
1:安装，不推荐使用cnpm可能会出现问题(我是出现了问题)。
#npm install -g cordova
2：检测cordova是否安装成功： 
#cordova -v
3：创建应用
#cordova create oneApp

vue创建应用

1：vue-cli(vue2)创建模版：这里是在cordova创建的应用目录中执行下面操作
#vue init webpack vueApp
#cd vueApp
#npm install 
#npm run dev

插曲:oneApp目录结构展示(这里只对文件夹具体看构建)
-hooks
-res
-platforms
-vueApp
-plugins
-www
2：修改vueApp项目中config\index.js
修改前- build: {
    // Template for index.html
    index: path.resolve(__dirname, '../dist/index.html'),

    // Paths
    assetsRoot: path.resolve(__dirname, '../dist'),
    assetsSubDirectory: 'static',
    assetsPublicPath: '/',
修改后- build: {
    // Template for index.html
    index: path.resolve(__dirname, '../../www/index.html'),

    // Paths
    assetsRoot: path.resolve(__dirname, '../../www'),
    assetsSubDirectory: 'static',
    assetsPublicPath: '',
3:执行npm run build 把文件通过webpack生成到oneApp的-www文件中

.apk生成（我自己只做了Android）

1：添加平台

$ cordova platform add ios
$ cordova platform add android

2：检查您是否满足构建平台的要求

$ cordova requirements
Requirements check results for android:
Java JDK: installed .
Android SDK: installed
Android target: installed android-19,android-21,android-22,android-23,Google Inc.:Google APIs:19,Google Inc.:Google APIs (x86 System Image):19,Google Inc.:Google APIs:23
Gradle: installed

Requirements check results for ios:
Apple OS X: not installed
Cordova tooling for iOS requires Apple OS X
Error: Some of requirements check failed

3：运行以下命令为所有平台构建项目
我这里自己只安装了Android，cordova build执行就ok
cordova build Android
cordova build ios

我的开发环境如下（仅供参考）

node:v6.11.4
npm:5.4.2
cordova：8.1.2 
jdk：1.8.0_191(java -version)
gradle：4.10.2
android sdk

cordova build Android遇到一个问题

Error:Could not find intellij-core.jar 
(com.android.tools.external.com-intellij:intellij-core:26.0.1).
 Searched in the following locations:
 https://jcenter.bintray.com/com/android/tools/external/com-intellij/
intellij-core/26.0.1/intellij-core-26.0.1.jar
这里访问谷歌[https://stackoverflow.com/questions/52946371/android-studio-could-not-find-intellij-core-jar](https://stackoverflow.com/questions/52946371/android-studio-could-not-find-intellij-core-jar "解决链接")

修改/platforms/android/CordovaLib/build.gradle文件
from this:
        repositories {
        jcenter()
        maven {
            url "https://maven.google.com"
        }
    }
to this:
        repositories {
        maven {
            url "https://maven.google.com"
        }
        jcenter()
    }
就ok了
