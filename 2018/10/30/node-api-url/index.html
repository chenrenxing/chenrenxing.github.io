url.parse

url.parse(urlString[, parseQueryString[, slashesDenoteHost]])

1:urlString <string> The URL string to parse.
2:parseQueryString 默认为false 如果为true解析出来的对象属性
    例如-query: { name: 'chenrenxing' } 为true
         -query: 'name=chenrenxing'   为false
3:slashesDenoteHost //coding.imooc.com/class/107.html?name=chenrenxing#yoxi'，true,true
如果为true可以进行解析不带协议的url，默认为false就不能解析不带协议的url

querystring.stringify（obj [，sep [，eq [，options]]]）

obj <Object>要序列化为URL查询字符串的对象
form - querystring.stringify({name:'keke',course:['hello','word'],form:';})
to - 'name=keke&course=hello&course=word&from='

sep <string>用于在查询字符串中分隔键和值对的子字符串。默认值： '&'。
form - querystring.stringify({name:'keke',course:['hello','word'],form:';},',')
to - 'name=keke,course=hello,course=word,from='

eq <string>。用于分隔查询字符串中的键和值的子字符串。默认值： '='。 
form - querystring.stringify({name:'keke',course:['hello','word'],form:';},',',':')
to - 'name:keke,course:hello,course:word,from:'
